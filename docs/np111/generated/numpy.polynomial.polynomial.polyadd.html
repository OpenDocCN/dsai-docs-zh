numpy.polynomial.polynomial.polyadd(c1, c2)[source]

给另一个多项式加一个多项式。

返回两个多项式c1 + c2的和。参数是从最低阶项到最高阶的系数序列，即[1,2,3]表示多项式1 + 2 * x + 3 * x ** 2。

参数：

c1，c2：array_like

1-D从低到高排序的多项式系数的数组。
返回：

out：ndarray

系数数组表示它们的和。

也可以看看

polysub，polymul，polydiv，polypow

例子

>>> from numpy.polynomial import polynomial as P
>>> c1 = (1,2,3)
>>> c2 = (3,2,1)
>>> sum = P.polyadd(c1,c2); sum
array([ 4.,  4.,  4.])
>>> P.polyval(2, sum) # 4 + 4(2) + 4(2**2)
28.0
