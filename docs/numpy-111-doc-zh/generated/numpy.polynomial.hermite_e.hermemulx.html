numpy.polynomial.hermite_e.hermemulx(c)[source]

将一个Hermite系列乘以x。

将Hermite系列c乘以x，其中x是自变量。

参数：

c：array_like

1 - D数组从低到高排列的Hermite系数系数。
返回：

out：ndarray

数组表示乘法的结果。

笔记

乘法使用形式的Hermite多项式的递归关系

xP_i（x）=（P_ {i + 1}（x）+ iP_ {i-1}（x）））

例子

>>> from numpy.polynomial.hermite_e import hermemulx
>>> hermemulx([1, 2, 3])
array([ 2.,  7.,  2.,  3.])
