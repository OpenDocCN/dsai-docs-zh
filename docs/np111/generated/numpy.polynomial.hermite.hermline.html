numpy.polynomial.hermite.hermline(off, scl)[source]

Hermite系列，其图形为直线。

参数：

off，scl：标量

指定的行由关闭 + scl * x给出。
返回：

y：ndarray

此模块表示关 + scl * x的Hermite系列。

也可以看看

polyline，chebline

例子

>>> from numpy.polynomial.hermite import hermline, hermval
>>> hermval(0,hermline(3, 2))
3.0
>>> hermval(1,hermline(3, 2))
5.0
