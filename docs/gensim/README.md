## 下载

### Docker

```
docker pull apachecn0/gensim-doc-zh
docker run -tid -p <port>:80 apachecn0/gensim-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install gensim-doc-zh
gensim-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g gensim-doc-zh
gensim-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
