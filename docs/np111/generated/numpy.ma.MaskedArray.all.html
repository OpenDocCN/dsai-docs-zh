MaskedArray.all(axis=None, out=None, keepdims=<class numpy._globals._NoValue>)[source]

如果所有元素均为True，则返回True。

输出数组被屏蔽，其中给定轴上的所有值都被屏蔽：如果输出将是标量并且所有值都被屏蔽，则输出masked。

有关完整文档，请参阅numpy.all。

例子

>>> np.ma.array([1,2,3]).all()
True
>>> a = np.ma.array([1,2,3], mask=True)
>>> (a.all() is np.ma.masked)
True
