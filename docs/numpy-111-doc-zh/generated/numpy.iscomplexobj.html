numpy.iscomplexobj

numpy.iscomplexobj(x)[source]

检查复杂类型或复数的数组。

检查输入的类型，而不是值。即使输入的虚部等于零，iscomplexobj也会计算为True。

参数：

x：any

输入可以是任何类型和形状。
返回：

iscomplexobj：bool

如果x是复杂类型或至少有一个复杂元素，则返回值True。

也可以看看

isrealobj，iscomplex

例子

>>> np.iscomplexobj(1)
False
>>> np.iscomplexobj(1+0j)
True
>>> np.iscomplexobj([3, 1+0j, True])
True
