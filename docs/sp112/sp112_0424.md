# `scipy.odr.OdrError`

```py
exception scipy.odr.OdrError
```

表示拟合过程中出现的异常。

当拟合过程中发生错误时，`odr` 会引发此异常。

```py
with_traceback()
```

Exception.with_traceback(tb) – 设置 self.__traceback__ 为 tb 并返回 self。
