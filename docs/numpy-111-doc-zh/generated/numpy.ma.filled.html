numpy.ma.filled(a, fill_value=None)[source]

将输入作为数组，将掩码数据替换为填充值。

如果a不是MaskedArray，则返回a本身。如果a为MaskedArray且fill_value为无，fill_value设置为a.fill_value

参数：

a：MaskedArray或array_like

输入对象。

fill_value：标量，可选

灌装值。默认值为None。
返回：

a：ndarray

填充数组。

也可以看看

compressed

例子

>>> x = np.ma.array(np.arange(9).reshape(3, 3), mask=[[1, 0, 0],
...                                                   [1, 0, 0],
...                                                   [0, 0, 0]])
>>> x.filled()
array([[999999,      1,      2],
       [999999,      4,      5],
       [     6,      7,      8]])
