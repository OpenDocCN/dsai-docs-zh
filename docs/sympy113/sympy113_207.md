量子力学相关的常数（如 hbar）。

```py
class sympy.physics.quantum.constants.HBar
```

数值和符号形式下的约化普朗克常数[[R755]](#r755)。

示例

```py
>>> from sympy.physics.quantum.constants import hbar
>>> hbar.evalf()
1.05457162000000e-34 
```
