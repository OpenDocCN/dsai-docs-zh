numpy.cosh

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.cosh(x[, out]) = <ufunc 'cosh'>

双曲余弦，元素。

等于1/2 * （np.exp（x） + np。 exp（-x））和np.cos(1j*x)。

参数：

x：array_like

输入数组。
返回：

out：ndarray

与x形状相同的输出数组。

例子

>>> np.cosh(0)
1.0

双曲余弦描述悬挂电缆的形状：

>>> import matplotlib.pyplot as plt
>>> x = np.linspace(-4, 4, 1000)
>>> plt.plot(x, np.cosh(x))
>>> plt.show()
