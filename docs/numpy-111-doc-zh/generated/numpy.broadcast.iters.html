broadcast.iters

沿着self的“组件”的迭代器的元组。

返回numpy.flatiter对象的元组，每个对象对应self的每个“组件”。

也可以看看

numpy.flatiter

例子

>>> x = np.array([1, 2, 3])
>>> y = np.array([[4], [5], [6]])
>>> b = np.broadcast(x, y)
>>> row, col = b.iters
>>> row.next(), col.next()
(1, 4)
