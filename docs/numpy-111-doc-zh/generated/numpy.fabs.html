numpy.fabs

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.fabs.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.fabs(x[, out]) = <ufunc 'fabs'>

按元素计算绝对值。

此函数返回x中数据的绝对值（正幅度）。不处理复杂值，使用absolute查找复杂数据的绝对值。

参数：

x：array_like

需要绝对值的数字的数组。如果x是标量，则结果y也将是标量。

out：ndarray，可选

数组，其中放置输出。它的类型被保留，并且它必须是保持输出的正确形状。请参阅doc.ufuncs。
返回：

y：ndarray或scalar

x的绝对值，返回值始终为浮点型。

也可以看看

absolute
绝对值，包括complex类型。

例子

>>> np.fabs(-1)
1.0
>>> np.fabs([-1.2, 1.2])
array([ 1.2,  1.2])
