class numpy.iinfo(type)[source]

整数类型的机器限制。

参数：

int_type：整数类型，dtype或实例

要获取有关信息的整数数据类型的种类。

也可以看看

finfo
浮点数据类型的等价物。

例子

类型：

>>> ii16 = np.iinfo(np.int16)
>>> ii16.min
-32768
>>> ii16.max
32767
>>> ii32 = np.iinfo(np.int32)
>>> ii32.min
-2147483648
>>> ii32.max
2147483647

使用实例：

>>> ii32 = np.iinfo(np.int32(10))
>>> ii32.min
-2147483648
>>> ii32.max
2147483647

属性

min 给定dtype的最小值。
max 给定dtype的最大值。
