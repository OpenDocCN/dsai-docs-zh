numpy.mirr(values, finance_rate, reinvest_rate)[source]

修正内部收益率。

参数：

值：array_like

返回现金流量（必须包含至少一个正值和一个负值）或nan。第一个值被认为是零时刻的沉没成本。

finance_rate：标量

支付现金流量的利率

reinvest_rate：scalar

再投资时的现金流量利率
返回：

out：float

修正内部收益率
