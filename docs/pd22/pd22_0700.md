# pandas.Index.item

> 原文：[`pandas.pydata.org/docs/reference/api/pandas.Index.item.html`](https://pandas.pydata.org/docs/reference/api/pandas.Index.item.html)

```py
Index.item()
```

返回基础数据的第一个元素作为 Python 标量。

返回：

标量

Series 或 Index 的第一个元素。

引发：

值错误

如果数据不是长度 = 1。

示例

```py
>>> s = pd.Series([1])
>>> s.item()
1 
```

对于一个索引：

```py
>>> s = pd.Series([1], index=['a'])
>>> s.index.item()
'a' 
```
