numpy.polynomial.polynomial.polyline(off, scl)[source]

返回表示线性多项式的数组。

参数：

off，scl：标量

线的“y截距”和“斜率”。
返回：

y：ndarray

此模块表示的线性多项式off + scl * x。

也可以看看

chebline

例子

>>> from numpy.polynomial import polynomial as P
>>> P.polyline(1,-1)
array([ 1, -1])
>>> P.polyval(1, P.polyline(1,-1)) # should be 0
0.0
