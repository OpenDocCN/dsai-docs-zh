matrix.any(axis=None, out=None)[source]

测试沿给定轴的任何数组元素是否为True。

有关完整文档，请参阅numpy.any。

参数：

axis：int，可选

执行逻辑或运算的轴

out：ndarray，可选

输出到现有数组而不是创建新数组，必须具有与预期输出相同的形状
返回：

任何：bool，ndarray

如果axis为None，则返回单个布尔值；否则返回ndarray
