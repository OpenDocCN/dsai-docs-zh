numpy.core.defchararray.greater(x1, x2)[source]

元素方式返回（x1> x2）。

与numpy.greater不同，此比较通过首先从字符串末尾剥除空格字符来执行。此行为用于与numarray的向后兼容性。

参数：

x1，x2：array_like的str或unicode

输入相同形状的数组。
返回：

out：ndarray或bool

输出bool的数组，如果x1和x2是标量，则为单个bool。
