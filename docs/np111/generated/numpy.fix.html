numpy.fix(x, y=None)[source]

向零舍入到最接近的整数。

将元素的浮点数组向零舍入到最接近的整数。舍入值作为浮点返回。

参数：

x：array_like

要舍入的浮点数组

y：ndarray，可选

输出数组
返回：

out：ndarray的浮点数

舍入数的数组

也可以看看

trunc，floor，ceil

around
舍入为给定的小数位数

例子

>>> np.fix(3.14)
3.0
>>> np.fix(3)
3.0
>>> np.fix([2.1, 2.9, -2.1, -2.9])
array([ 2.,  2., -2., -2.])
