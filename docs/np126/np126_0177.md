# numpy.dtype.num

> 原文：[`numpy.org/doc/1.26/reference/generated/numpy.dtype.num.html`](https://numpy.org/doc/1.26/reference/generated/numpy.dtype.num.html)

属性

```py
dtype.num
```

每种 21 种不同的内置类型都有一个唯一的数字。

这些大致按照从最不精确到最精确的顺序排列。

示例

```py
>>> dt = np.dtype(str)
>>> dt.num
19 
```

```py
>>> dt = np.dtype(float)
>>> dt.num
12 
```
