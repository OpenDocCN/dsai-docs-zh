```py
scipy.special.pdtrc(k, m, out=None) = <ufunc 'pdtrc'>
```

泊松生存函数

返回泊松分布的从 k+1 到无穷大项的总和：sum(exp(-m) * m**j / j!, j=k+1..inf) = gammainc( k+1, m)。参数必须为非负 double 类型。

参数：

**k**array_like

发生次数（非负实数）

**m**array_like

形状参数（非负实数）

**out**ndarray，可选

函数结果的可选输出数组

返回值：

标量或者 ndarray

泊松生存函数的值
