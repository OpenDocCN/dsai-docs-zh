MaskedArray.resize(newshape, refcheck=True, order=False)[source]

警告

此方法不执行任何操作，但抛出一个ValueError异常。被屏蔽的数组不拥有其数据，因此不能安全地调整大小。请改用numpy.ma.resize函数。

此方法难以安全实施，并且可能在未来版本的NumPy中被弃用。
