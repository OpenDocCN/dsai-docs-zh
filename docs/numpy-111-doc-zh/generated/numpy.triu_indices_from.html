numpy.triu_indices_from

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.triu_indices_from(arr, k=0)[source]

返回arr的上三角形的索引。

有关详细信息，请参阅triu_indices。

参数：

arr：ndarray，shape（N，N）

指数将对方数组有效。

k：int，可选

对角线偏移（详见triu）。
返回：

triu_indices_from：元组，ndarray的形状（2），形状（N）

arr的上三角形的索引。

也可以看看

triu_indices，triu
