numpy.polynomial.hermite_e.hermecompanion(c)[source]

返回c的缩放伴随矩阵。

基底多项式被缩放，使得当c是HermiteE基底多项式时，伴随矩阵是对称的。这提供比未缩放的情况更好的特征值估计，并且对于基本多项式，如果使用numpy.linalg.eigvalsh来获得它们，则特征值被保证为实数。

参数：

c：array_like

1〜D数组HermiteE系数系数从低到高排列。
返回：

mat：ndarray

尺寸（deg，deg）的缩放伴随矩阵。
