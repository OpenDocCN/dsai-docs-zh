numpy.random.exponential

校对：（虚位以待）

numpy.random.exponential(scale=1.0, size=None)

从指数分布绘制样本。

其概率密度函数为

对于x > 0和其他地方为0。是比例参数，它是速率参数的倒数。速率参数是指数分布[R218]的备选的广泛使用的参数化。

指数分布是几何分布的连续模拟。它描述了许多常见的情况，例如在许多暴雨[R216]上测量的雨滴的大小，或者在请求到维基百科[R217]的页面之间的时间。

参数：

scale：float

缩放参数，。

size：int或tuple的整数，可选

输出形状。如果给定形状是例如（m， n， k），则 m * n * k默认值为None，在这种情况下返回单个值。
