# scipy.special.chndtr

非中心卡方累积分布函数

累积分布函数由以下给出：

\[P(\chi^{\prime 2} \vert \nu, \lambda) =\sum_{j=0}^{\infty} e^{-\lambda /2} \frac{(\lambda /2)^j}{j!} P(\chi^{\prime 2} \vert \nu + 2j),\]

其中 \(\nu > 0\) 是自由度（`df`），\(\lambda \geq 0\) 是非中心参数（`nc`）。

参数：

**x**array_like

积分的上限；必须满足 `x >= 0`

**df**array_like

自由度；必须满足 `df > 0`

**nc**array_like

非中心参数；必须满足 `nc >= 0`

**out**ndarray，可选

函数结果的可选输出数组

返回：

**x**标量或数组

非中心卡方累积分布函数的值。
