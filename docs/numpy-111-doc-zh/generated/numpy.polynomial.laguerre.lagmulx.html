numpy.polynomial.laguerre.lagmulx(c)[source]

将Laguerre系列乘以x。

将Laguerre系列c乘以x，其中x是自变量。

参数：

c：array_like

Laguerre系数的1-D数组从低到高排序。
返回：

out：ndarray

数组表示乘法的结果。

笔记

乘法使用形式中的Laguerre多项式的递归关系

xP_i（x）=（ - （i + 1）* P_ {i + 1}（x）+（2i + 1）P_ {i}

例子

>>> from numpy.polynomial.laguerre import lagmulx
>>> lagmulx([1, 2, 3])
array([ -1.,  -1.,  11.,  -9.])
