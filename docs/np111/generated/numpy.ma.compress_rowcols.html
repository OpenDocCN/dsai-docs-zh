numpy.ma.compress_rowcols

校对：（虚位以待）

numpy.ma.compress_rowcols(x, axis=None)[source]

抑制包含屏蔽值的2-D数组的行和/或列。

抑制行为通过axis参数选择。

  • 如果axis为None，则行和列都被抑制。
  • 如果axis为0，则只抑制行。
  • 如果axis为1或-1，则只抑制列。
参数：

x：array_like，MaskedArray

数组要操作。如果不是MaskedArray实例（或者如果没有数组元素被屏蔽），则x被解释为MaskedArray，其中掩码设置为nomask。必须是2D数组。

axis：int，可选

沿其执行操作的轴。默认值为None。
返回：

compressed_array：ndarray

压缩的数组。

例子

>>> x = np.ma.array(np.arange(9).reshape(3, 3), mask=[[1, 0, 0],
...                                                   [1, 0, 0],
...                                                   [0, 0, 0]])
>>> x
masked_array(data =
 [[-- 1 2]
 [-- 4 5]
 [6 7 8]],
             mask =
 [[ True False False]
 [ True False False]
 [False False False]],
       fill_value = 999999)

>>> np.ma.compress_rowcols(x)
array([[7, 8]])
>>> np.ma.compress_rowcols(x, 0)
array([[6, 7, 8]])
>>> np.ma.compress_rowcols(x, 1)
array([[1, 2],
       [4, 5],
       [7, 8]])
