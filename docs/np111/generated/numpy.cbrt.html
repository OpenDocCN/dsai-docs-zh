numpy.cbrt

校对：（虚位以待）

numpy.cbrt(x[, out]) = <ufunc 'cbrt'>

以元素方式返回数组的多维数据集根。

版本1.10.0中的新功能。

参数：

x：array_like

需要立方根的值。

out：ndarray，可选

替代数组对象，其中放置结果；如果提供，它必须具有与x相同的形状
返回：

y：ndarray

与x形状相同的数组，包含x中每个元素的立方体立方根。如果提供out，则y是对它的引用。

例子

>>> np.cbrt([1,8,27])
array([ 1.,  2.,  3.])
