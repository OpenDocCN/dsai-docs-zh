numpy.core.defchararray.add(x1, x2)[source]

返回逐元素的字符串连接为海峡或Unicode 2数组。

数组x1和x2必须具有相同的形状。

参数：

x1：array_like of str或unicode

输入数组。

x2：array_like的str或unicode

输入数组。
返回：

添加：ndarray

输出数组string_或unicode_，取决于与x1和x2相同形状的输入类型。
