numpy.random.chisquare(df, size=None)

从卡方分布绘制样本。

当df独立随机变量，每个具有标准正态分布（平均值0，方差1），被平方和相加，得到的分布是卡方（见注释）。这种分布通常用于假设检验。

参数：

df：int

自由度数。

size：int或tuple的整数，可选

输出形状。如果给定形状是例如（m， n， k），则 m * n * k默认值为None，在这种情况下返回单个值。
返回：

输出：ndarray

从分布绘制的样本，包装在大小形数组中。
