numpy.c_ = <numpy.lib.index_tricks.CClass object>

将切片对象转换为沿第二轴的连接。

这是np.r _ [' - 1,2,0'， 索引 表达式] ，这是有用的，因为其常见的发生。特别地，数组在被升级到至少2-D之后将沿着它们的最后轴被堆叠，其中1被后置为形状（由1-D数组组成的列向量）。

有关详细文档，请参见r_。

例子

>>> np.c_[np.array([[1,2,3]]), 0, 0, np.array([[4,5,6]])]
array([[1, 2, 3, 0, 0, 4, 5, 6]])
