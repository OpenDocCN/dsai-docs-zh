# pandas.Timedelta.days

> 原文：[`pandas.pydata.org/docs/reference/api/pandas.Timedelta.days.html`](https://pandas.pydata.org/docs/reference/api/pandas.Timedelta.days.html)

```py
Timedelta.days
```

返回时间间隔的天数。

返回：

int

示例

```py
>>> td = pd.Timedelta(1, "d")
>>> td.days
1 
```

```py
>>> td = pd.Timedelta('4 min 3 us 42 ns')
>>> td.days
0 
```
