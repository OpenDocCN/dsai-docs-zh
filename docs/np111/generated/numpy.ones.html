numpy.ones(shape, dtype=None, order='C')[source]

返回给定形状和类型的新数组，用数字填充。

参数：

shape：int或ints序列

新数组的形状，例如（2， 3）或2。

dtype：数据类型，可选

数组的所需数据类型，例如numpy.int8。默认值为numpy.float64。

order：{'C'，'F'}，可选

是否在存储器中以C或Fortran连续（按行或列方式）存储多维数据。
返回：

out：ndarray

具有给定形状，dtype和顺序的数组。

也可以看看

zeros，ones_like

例子

>>> np.ones(5)
array([ 1.,  1.,  1.,  1.,  1.])

>>> np.ones((5,), dtype=np.int)
array([1, 1, 1, 1, 1])

>>> np.ones((2, 1))
array([[ 1.],
       [ 1.]])

>>> s = (2,2)
>>> np.ones(s)
array([[ 1.,  1.],
       [ 1.,  1.]])
