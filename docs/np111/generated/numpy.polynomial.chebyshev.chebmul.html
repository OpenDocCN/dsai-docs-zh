numpy.polynomial.chebyshev.chebmul(c1, c2)[source]

将一个切比雪夫系列乘以另一个。

返回两个切比雪夫系列c1 * c2的乘积。参数是从最低阶“项”到最高的系数序列，例如[1,2,3]表示系列T_0 + 2 * T_1 + 3 * T_2。

参数：

c1，c2：array_like

1-D数组的切比雪夫系数系数从低到高排序。
返回：

out：ndarray

切比雪夫系数系数代表他们的产品。

也可以看看

chebadd，chebsub，chebdiv，chebpow

笔记

通常，两个C系列的（多项式）乘积导致不在Chebyshev多项式基组中的项。因此，为了将产品表示为C系列，通常有必要将产品“再投影”到所述基础集合上，这通常产生“非直观的”（但是正确的）结果；请参阅下面的示例部分。

例子

>>> from numpy.polynomial import chebyshev as C
>>> c1 = (1,2,3)
>>> c2 = (3,2,1)
>>> C.chebmul(c1,c2) # multiplication requires "reprojection"
array([  6.5,  12. ,  12. ,   4. ,   1.5])
