numpy.polynomial.polynomial.polymul(c1, c2)[source]

将一个多项式乘以另一个。

返回两个多项式c1 * c2的乘积。参数是从最低阶项到最高阶的系数序列，例如，[1,2,3]表示多项式1 + x + 3 * x ** 2.

参数：

c1，c2：array_like

1-D数组的系数，表示相对于“标准”基础的多项式，并且从最低阶项到最高阶排序。
返回：

out：ndarray

它们的乘积的系数。

也可以看看

polyadd，polysub，polydiv，polypow

例子

>>> from numpy.polynomial import polynomial as P
>>> c1 = (1,2,3)
>>> c2 = (3,2,1)
>>> P.polymul(c1,c2)
array([  3.,   8.,  14.,   8.,   3.])
