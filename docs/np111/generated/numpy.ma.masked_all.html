numpy.ma.masked_all(shape, dtype=<type 'float'>)[source]

与被掩没的所有元素的空的被掩没的数组。

返回给定形状和dtype的空屏蔽数组，其中所有数据都被屏蔽。

参数：

shape：tuple

所需MaskedArray的形状。

dtype：dtype，可选

输出的数据类型。
返回：

a：MaskedArray

所有数据被屏蔽的屏蔽数组。

也可以看看

masked_all_like
在现有数组上建模的空掩码数组。

例子

>>> import numpy.ma as ma
>>> ma.masked_all((3, 3))
masked_array(data =
 [[-- -- --]
 [-- -- --]
 [-- -- --]],
      mask =
 [[ True  True  True]
 [ True  True  True]
 [ True  True  True]],
      fill_value=1e+20)

dtype参数定义基础数据类型。

>>> a = ma.masked_all((3, 3))
>>> a.dtype
dtype('float64')
>>> a = ma.masked_all((3, 3), dtype=np.int32)
>>> a.dtype
dtype('int32')
