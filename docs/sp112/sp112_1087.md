# scipy.special.cbrt

> 原文链接：[https://docs.scipy.org/doc/scipy-1.12.0/reference/generated/scipy.special.cbrt.html#scipy.special.cbrt](https://docs.scipy.org/doc/scipy-1.12.0/reference/generated/scipy.special.cbrt.html#scipy.special.cbrt)

```py
scipy.special.cbrt(x, out=None) = <ufunc 'cbrt'>
```

*x* 的逐元素立方根。

参数：

**x**array_like

*x* 必须包含实数。

**out**ndarray，可选

函数值的可选输出数组

返回值：

标量或者 ndarray

*x* 中每个值的立方根。

示例

```py
>>> from scipy.special import cbrt 
```

```py
>>> cbrt(8)
2.0
>>> cbrt([-8, -3, 0.125, 1.331])
array([-2\.        , -1.44224957,  0.5       ,  1.1       ]) 
```
