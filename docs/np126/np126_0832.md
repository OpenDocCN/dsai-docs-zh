# numpy.around

> 原文：[`numpy.org/doc/1.26/reference/generated/numpy.around.html`](https://numpy.org/doc/1.26/reference/generated/numpy.around.html)

```py
numpy.around(a, decimals=0, out=None)
```

将数组四舍五入到指定的小数位数。

`around` 是 `round` 的别名。

另请参阅

`ndarray.round`

等效方法

`round`
