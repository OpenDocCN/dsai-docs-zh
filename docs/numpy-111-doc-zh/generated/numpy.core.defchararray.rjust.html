numpy.core.defchararray.rjust(a, width, fillchar=' ')[source]

返回一个数组，其元素为a在长度宽度的字符串中右对齐。

单元调用str.rjust。

参数：

a：array_like of str或unicode

width：int

生成的字符串的长度

fillchar：str或unicode，可选

用于填充的字符
返回：

out：ndarray

输出str或unicode的数组，取决于输入类型
