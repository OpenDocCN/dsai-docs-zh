MaskedArray.ids()[source]

返回数据和掩码区域的地址。

参数：无

例子

>>> x = np.ma.array([1, 2, 3], mask=[0, 1, 1])
>>> x.ids()
(166670640, 166659832)

如果数组没有掩码，则返回nomask的地址。此地址通常不接近内存中的数据：

>>> x = np.ma.array([1, 2, 3])
>>> x.ids()
(166691080, 3083169284L)
