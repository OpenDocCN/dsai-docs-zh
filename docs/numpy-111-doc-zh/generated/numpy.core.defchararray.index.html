numpy.core.defchararray.index(a, sub, start=0, end=None)[source]

像find，但在找不到子字符串时引发ValueError。

逐个调用str.index。

参数：

a：array_like of str或unicode

sub：str或unicode

start，end：int，可选
返回：

out：ndarray

输出数组的整数。如果未找到sub，则返回-1。
