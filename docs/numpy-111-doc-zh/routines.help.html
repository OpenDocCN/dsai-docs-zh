NumPy-specific help functions

Finding help

lookfor（what [，module，import_modules，...]） 在docstrings上执行关键字搜索。

Reading help

info（[object，maxwidth，output，toplevel]） 获取函数，类或模块的帮助信息。
source（object [，output]） 打印或写入文件Numpy对象的源代码。
