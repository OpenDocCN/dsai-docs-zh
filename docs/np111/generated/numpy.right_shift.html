numpy.right_shift

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.right_shift.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.right_shift(x1, x2[, out]) = <ufunc 'right_shift'>

将整数的位向右移位。

位向右移动x2。因为数字的内部表示是二进制格式，所以此操作等效于将x1除以2**x2。

参数：

x1：array_like，int

输入值。

x2：array_like，int

x1右侧要删除的位数。
返回：

out：ndarray，int

返回x1，位向右移位x2次。

也可以看看

left_shift
将整数的位向左移位。
binary_repr
返回输入号码的二进制表示为字符串。

例子

>>> np.binary_repr(10)
'1010'
>>> np.right_shift(10, 1)
5
>>> np.binary_repr(5)
'101'

>>> np.right_shift(10, [1,2,3])
array([5, 2, 1])
