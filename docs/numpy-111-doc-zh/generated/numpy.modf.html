numpy.modf

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.modf.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.modf(x[, out1, out2]) = <ufunc 'modf'>

以元素方式返回数组的小数和整数部分。

如果给定数为负数，分数和整数部分为负。

参数：

x：array_like

输入数组。
返回：

y1：ndarray

x的小数部分。

y2：ndarray

x的积分部分。

笔记

对于整数输入，返回值是浮点数。

例子

>>> np.modf([0, 3.5])
(array([ 0. ,  0.5]), array([ 0.,  3.]))
>>> np.modf(-0.5)
(-0.5, -0)
