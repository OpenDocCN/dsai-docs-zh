校对：（虚位以待）

RandomState.rand(d0, d1, ..., dn)

给定形状中的随机值。

创建给定形状的数组并使用来自[0， t> 1）的均匀分布的随机样本填充它。

参数：

d0，d1，...，dn：int，可选

返回的数组的尺寸应该都是正数。如果没有给出参数，则返回单个Python浮动。
返回：

out：ndarray，shape （d0， d1， ...， dn）

随机值。

也可以看看

random

笔记

这是一个方便的功能。如果你想要一个采用形状元组作为第一个参数的接口，请参考np.random.random_sample。

例子

>>> np.random.rand(3,2)
array([[ 0.14022471,  0.96360618],  #random
       [ 0.37601032,  0.25528411],  #random
       [ 0.49313049,  0.94909878]]) #random
