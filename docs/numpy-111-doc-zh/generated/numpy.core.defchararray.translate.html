numpy.core.defchararray.translate(a, table, deletechars=None)[source]

对于a中的每个元素，返回一个字符串的副本，其中可选参数deletechars中出现的所有字符都将被删除，其余字符已通过给定的转换表。

逐个调用str.translate。

参数：

a：数组类似str或unicode

表：长度为256的str

deletechars：str
返回：

out：ndarray

输出str或unicode的数组，取决于输入类型
