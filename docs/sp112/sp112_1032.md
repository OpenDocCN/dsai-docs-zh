第二类椭圆球面径向函数及其导数

计算第二类椭圆球面径向函数及其导数（关于 *x*），对于模参数 m>=0 和 n>=m，椭圆球面参数 *c* 和 `|x| < 1.0`。

参数：

**m** 数组或类似对象

非负模参数 m

**n** 数组或类似对象

模参数 n (>= m)

**c** 数组或类似对象

椭圆球面参数

**cv** 数组或类似对象

特征值

**x** 数组或类似对象

实参数 (`|x| < 1.0`)

**out** ndarray，可选

可选输出数组，用于函数结果

返回：

**s** 标量或 ndarray

函数的值

**sp** 标量或 ndarray

导数的值 vs x
