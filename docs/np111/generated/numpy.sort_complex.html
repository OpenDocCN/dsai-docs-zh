numpy.sort_complex

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.sort_complex.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.sort_complex(a)[source]

使用实部，然后是虚部对复数数组进行排序。

参数：

a：array_like

输入数组
返回：

out：complex ndarray

始终返回已排序的复杂数组。

例子

>>> np.sort_complex([5, 3, 6, 2, 1])
array([ 1.+0.j,  2.+0.j,  3.+0.j,  5.+0.j,  6.+0.j])

>>> np.sort_complex([1 + 2j, 2 - 1j, 3 - 2j, 3 - 3j, 3 + 5j])
array([ 1.+2.j,  2.-1.j,  3.-3.j,  3.-2.j,  3.+5.j])
