numpy.ma.mr_ = <numpy.ma.extras.mr_class object>

将切片对象转换为沿第一轴的连接。

这是lib.index_tricks.RClass的掩码数组版本。

也可以看看

lib.index_tricks.RClass

例子

>>> np.ma.mr_[np.ma.array([1,2,3]), 0, 0, np.ma.array([4,5,6])]
array([1, 2, 3, 0, 0, 4, 5, 6])
