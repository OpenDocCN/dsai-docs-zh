numpy.random.rand

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.random.rand.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.random.rand(d0, d1, ..., dn)

给定形状的随机值。

创建一个给定形状的数组，并使用来自[0, 1)的均匀分布的随机样本填充它。

参数：

d0，d1，...，dn：int，可选

返回的数组的尺寸应该都是正数。如果没有给出参数，则返回一个单独的Python浮点数。
返回：

out : ndarray, shape (d0, d1, ..., dn)

随机值。

另见

random

注

这是一个便捷函数。如果你想要一个接受一个形状元组作为第一个参数的接口，请参考np.random.random_sample。

例子

>>> np.random.rand(3,2)
array([[ 0.14022471,  0.96360618],  #random
       [ 0.37601032,  0.25528411],  #random
       [ 0.49313049,  0.94909878]]) #random
