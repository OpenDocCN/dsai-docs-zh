numpy.polynomial.laguerre.lag2poly(c)[source]

将Laguerre系列转换为多项式。

将表示从最低程度到最高程度排序的Laguerre系数的系数的数组转换为从最低到最高程度排序的等效多项式的系数（相对于“标准”基础）的数组。

参数：

c：array_like

包含Laguerre系数系数的1-D数组，从最低阶到最高阶排序。
返回：

pol：ndarray

1-D数组，其包含从最低次项到最高次序排序的等效多项式的系数（相对于“标准”基础）。

笔记

在多项式基组之间进行转换的简单方法是使用类实例的convert方法。

例子

>>> from numpy.polynomial.laguerre import lag2poly
>>> lag2poly([ 23., -63.,  58., -18.])
array([ 0.,  1.,  2.,  3.])
