numpy.getbuffer

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.getbuffer.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.getbuffer(obj[, offset[, size]])

从给定对象创建一个缓冲对象，引用从偏移开始的长度大小的切片。

默认是整个缓冲区。尝试读写缓冲区后跟随只读缓冲区。

参数：

obj：object

offset：int，可选

size：int，可选
返回：

buffer_obj：buffer

例子

>>> buf = np.getbuffer(np.ones(5), 1, 3)
>>> len(buf)
3
>>> buf[0]
'\x00'
>>> buf
<read-write buffer for 0x8af1e70, size 3, offset 1 at 0x8ba4ec0>
