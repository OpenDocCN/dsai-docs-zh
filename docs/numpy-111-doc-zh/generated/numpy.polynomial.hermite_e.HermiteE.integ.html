HermiteE.integ(m=1, k=[], lbnd=None)[source]

整合。

返回一个系列实例，它是当前系列的定积分。

参数：

m：非负整数

要执行的集成数。

k：array_like

积分常数。第一常数应用于第一积分，第二常数应用于第二积分，等等。值列表的长度必须小于或等于m，任何缺失值都设置为零。

lbnd：标量

定积分的下限。
返回：

new_series：series

代表积分的新系列。域与集成系列的域相同。
