numpy.polymul(a1, a2)[source]

找到两个多项式的乘积。

查找从两个输入多项式相乘得到的多项式。每个输入必须是poly1d对象或多项式系数的1D序列，从最高到最低度。

参数：

a1，a2：array_like或poly1d对象

输入多项式。
返回：

out：ndarray或poly1d对象

由输入的乘法产生的多项式。如果任一输入是poly1d对象，则输出也是poly1d对象。否则，它是从最高到最低程度的多项式系数的1D数组。

也可以看看

poly1d
一维多项式类。

poly, polyadd, polyder, polydiv, polyfit, polyint, polysub, polyval

convolve
数组卷积。与polymul相同的输出，但具有重叠模式的参数。

例子

>>> np.polymul([1, 2, 3], [9, 5, 1])
array([ 9, 23, 38, 17,  3])

使用poly1d对象：

>>> p1 = np.poly1d([1, 2, 3])
>>> p2 = np.poly1d([9, 5, 1])
>>> print(p1)
   2
1 x + 2 x + 3
>>> print(p2)
   2
9 x + 5 x + 1
>>> print(np.polymul(p1, p2))
   4      3      2
9 x + 23 x + 38 x + 17 x + 3
