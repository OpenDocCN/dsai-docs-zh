numpy.expm1(x[, out]) = <ufunc 'expm1'>

对数组中的所有元素计算exp（x） - 1

参数：

x：array_like

输入值。
返回：

out：ndarray

单元指数减1：out = exp（x） - / t5>。

笔记

This function provides greater precision than exp(x) - 1 for small values of x.

例子

exp（1e-10） - 1的真值是1.00000000005e-10这个例子显示了expm1在这种情况下的优越性。

>>> np.expm1(1e-10)
1.00000000005e-10
>>> np.exp(1e-10) - 1
1.000000082740371e-10
