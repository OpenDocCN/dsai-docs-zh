numpy.identity(n, dtype=None)[source]

返回身份数组。

身份数组是一个正方形数组，在主对角线上有一个数组。

参数：

n：int

n x n输出中的行（和列）数。

dtype：数据类型，可选

输出的数据类型。默认为float。
返回：

out：ndarray

n x n数组，其主对角线设置为1，所有其他元素为0。

例子

>>> np.identity(3)
array([[ 1.,  0.,  0.],
       [ 0.,  1.,  0.],
       [ 0.,  0.,  1.]])
