numpy.ma.size(obj, axis=None)[source]

返回给定轴上的元素数。

参数：

a：array_like

输入数据。

axis：int，可选

对元素计数的轴。默认情况下，给出元素的总数。
返回：

element_count：int

沿指定轴的元素数。

也可以看看

shape
数组的维数
ndarray.shape
数组的维数
ndarray.size
数组中的元素数

例子

>>> a = np.array([[1,2,3],[4,5,6]])
>>> np.size(a)
6
>>> np.size(a,1)
3
>>> np.size(a,0)
2
