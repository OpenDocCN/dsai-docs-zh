校对：（虚位以待）

recarray.dot(b, out=None)

两个数组的点积。

有关完整文档，请参阅numpy.dot。

也可以看看

numpy.dot
等效函数

例子

>>> a = np.eye(2)
>>> b = np.ones((2, 2)) * 2
>>> a.dot(b)
array([[ 2.,  2.],
       [ 2.,  2.]])

这个数组方法可以方便地链接：

>>> a.dot(b).dot(b)
array([[ 8.,  8.],
       [ 8.,  8.]])
