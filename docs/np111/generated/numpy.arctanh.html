numpy.arctanh(x[, out]) = <ufunc 'arctanh'>

逆双曲正切元素。

参数：

x：array_like

输入数组。
返回：

out：ndarray

数组的形状与x相同。

也可以看看

emath.arctanh

笔记

arctanh是多值函数：对于每个x，存在无限多个数字z，使得tanh（z）= x t5 >。约定是返回其虚部在[ - pi / 2，pi / 2]中的z。

对于实值输入数据类型，arctanh始终返回实际输出。对于不能表示为实数或无穷大的每个值，它会产生nan并设置无效浮点错误标志。

对于复值输入，arctanh是具有分支切口[ - 1，-inf]和[1，inf]的复杂分析函数，并且在前者上从上方连续，在后者上从下方连续。

反双曲正切也称为atanh或tanh^-1。

例子

>>> np.arctanh([0, -0.5])
