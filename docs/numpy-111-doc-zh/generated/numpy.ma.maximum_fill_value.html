numpy.ma.maximum_fill_value(obj)[source]

返回可由对象的dtype表示的最小值。

此函数适用于计算适合获取带有给定dtype的数组的最大值的填充值。

参数：

obj：{ndarray，dtype}

可以查询其为数值类型的对象。
返回：

val：标量

最小可表示值。
上升：

TypeError

如果obj不是合适的数值类型。

也可以看看

minimum_fill_value
反函数。
set_fill_value
设置掩码数组的填充值。
MaskedArray.fill_value
返回当前填充值。

例子

>>> import numpy.ma as ma
>>> a = np.int8()
>>> ma.maximum_fill_value(a)
-128
>>> a = np.int32()
>>> ma.maximum_fill_value(a)
-2147483648

也可以传递数字数据的数组。

>>> a = np.array([1, 2, 3], dtype=np.int8)
>>> ma.maximum_fill_value(a)
-128
>>> a = np.array([1, 2, 3], dtype=np.float32)
>>> ma.maximum_fill_value(a)
-inf
