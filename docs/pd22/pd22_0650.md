# `pandas.Index.dtype`

> 原文：[`pandas.pydata.org/docs/reference/api/pandas.Index.dtype.html`](https://pandas.pydata.org/docs/reference/api/pandas.Index.dtype.html)

```py
Index.dtype
```

返回底层数据的 dtype 对象。

Examples

```py
>>> idx = pd.Index([1, 2, 3])
>>> idx
Index([1, 2, 3], dtype='int64')
>>> idx.dtype
dtype('int64') 
```
