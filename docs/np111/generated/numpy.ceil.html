numpy.ceil

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.ceil.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.ceil(x[, out]) = <ufunc 'ceil'>

元素方式返回输入的上限。

标量x的单元是最小整数i，使得i> = x。它通常表示为。

参数：

x：array_like

输入数据。
返回：

y：ndarray或scalar

x中每个元素的上限，其中float dtype。

也可以看看

floor，trunc，rint

例子

>>> a = np.array([-1.7, -1.5, -0.2, 0.2, 1.5, 1.7, 2.0])
>>> np.ceil(a)
array([-1., -1., -0.,  1.,  2.,  2.,  2.])
