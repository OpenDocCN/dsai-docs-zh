numpy.core.defchararray.mod(a, values)[source]

返回（a％i），这是Python 2.6之前的字符串格式化（iterpolation），对于一个array_likes的str或unicode的元素。

参数：

a：array_like of str或unicode

值：array_like的值

这些值将以元素方式内插到字符串中。
返回：

out：ndarray

输出str或unicode的数组，取决于输入类型
