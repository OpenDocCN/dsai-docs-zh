MaskedArray.byteswap(inplace)

交换数组元素的字节

通过返回一个字节交换的数组，可选地在位置交换，在低端和大端数据表示之间切换。

参数：

inplace：bool，可选

如果True，则就地交换字节，默认值为False。
返回：

out：ndarray

字节交换数组。如果inplace是True，则这是一个视图。

例子

>>> A = np.array([1, 256, 8755], dtype=np.int16)
>>> map(hex, A)
['0x1', '0x100', '0x2233']
>>> A.byteswap(True)
array([  256,     1, 13090], dtype=int16)
>>> map(hex, A)
['0x100', '0x1', '0x3322']

数组的字符串不交换

>>> A = np.array(['ceg', 'fac'])
>>> A.byteswap()
array(['ceg', 'fac'],
      dtype='|S3')
