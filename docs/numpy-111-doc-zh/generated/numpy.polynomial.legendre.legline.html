numpy.polynomial.legendre.legline(off, scl)[source]

Legendre系列的图形是一条直线。

参数：

off，scl：标量

指定的行由关闭 + scl * x给出。
返回：

y：ndarray

此模块表示了关 + scl * x的Legendre系列。

也可以看看

polyline，chebline

例子

>>> import numpy.polynomial.legendre as L
>>> L.legline(3,2)
array([3, 2])
>>> L.legval(-3, L.legline(3,2)) # should be -3
-3.0
