SciPy 1.3.3 是一个修复漏洞的版本，与 1.3.2 相比没有新功能。特别是为了解决 Windows 和 Python 3.8 在 macOS 上涉及多进程的测试套件问题。

Wheels 也已更新，以在适当位置放置 msvcp140.dll，此前曾引发问题。
