MaskedArray.compressed()[source]

将所有非屏蔽数据作为1-D数组。

返回：

data：ndarray

返回保存非掩码数据的新ndarray。

笔记

结果是不 a MaskedArray！

例子

>>> x = np.ma.array(np.arange(5), mask=[0]*2 + [1]*3)
>>> x.compressed()
array([0, 1])
>>> type(x.compressed())
<type 'numpy.ndarray'>
