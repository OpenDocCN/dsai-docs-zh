numpy.ma.mask_cols

校对：（虚位以待）

numpy.ma.mask_cols(a, axis=None)[source]

屏蔽包含屏蔽值的2D数组的列。

此函数是mask_rowcols的快捷方式，轴等于1。

也可以看看

mask_rowcols
屏蔽2D数组的行和/或列。
masked_where
满足条件的掩码。

例子

>>> import numpy.ma as ma
>>> a = np.zeros((3, 3), dtype=np.int)
>>> a[1, 1] = 1
>>> a
array([[0, 0, 0],
       [0, 1, 0],
       [0, 0, 0]])
>>> a = ma.masked_equal(a, 1)
>>> a
masked_array(data =
 [[0 0 0]
 [0 -- 0]
 [0 0 0]],
      mask =
 [[False False False]
 [False  True False]
 [False False False]],
      fill_value=999999)
>>> ma.mask_cols(a)
masked_array(data =
 [[0 -- 0]
 [0 -- 0]
 [0 -- 0]],
      mask =
 [[False  True False]
 [False  True False]
 [False  True False]],
      fill_value=999999)
