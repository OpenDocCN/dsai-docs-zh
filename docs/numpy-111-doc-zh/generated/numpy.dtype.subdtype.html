dtype.subdtype

Tuple (item_dtype, shape) if this dtype describes a sub-array, and None otherwise.

形状是此数据类型描述的子数组的固定形状，item_dtype数组的数据类型。

如果检索到dtype对象具有此属性的字段，则shape隐含的额外维度将粘贴到检索到的数组的末尾。
