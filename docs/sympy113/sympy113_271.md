# 全息函数

> 原文：[`docs.sympy.org/latest/modules/holonomic/index.html`](https://docs.sympy.org/latest/modules/holonomic/index.html)

`holonomic` 模块旨在处理全息函数及其诸如加法、乘法、组合、积分和微分等各种操作。该模块还实现了各种类型的转换，例如将全息函数转换为不同形式及其逆操作。
