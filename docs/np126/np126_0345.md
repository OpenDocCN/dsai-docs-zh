# numpy.broadcast

> 原文：[`numpy.org/doc/1.26/reference/generated/numpy.broadcast.html`](https://numpy.org/doc/1.26/reference/generated/numpy.broadcast.html)

```py
class numpy.broadcast
```

生成一个模拟广播的对象。

参数:

**in1, in2, …**array_like

输入参数。

返回:

**b**广播对象

将输入参数相互广播，并返回封装结果的对象。 其中包括`形状`和`nd`属性，并且可以用作迭代器。

另请参阅

`broadcast_arrays`

`broadcast_to`

`广播形状`

示例

手动添加两个向量，使用广播:

```py
>>> x = np.array([[1], [2], [3]])
>>> y = np.array([4, 5, 6])
>>> b = np.broadcast(x, y) 
```

```py
>>> out = np.empty(b.shape)
>>> out.flat = [u+v for (u,v) in b]
>>> out
array([[5.,  6.,  7.],
 [6.,  7.,  8.],
 [7.,  8.,  9.]]) 
```

与内置广播进行比较:

```py
>>> x + y
array([[5, 6, 7],
 [6, 7, 8],
 [7, 8, 9]]) 
```

属性:

`索引`

广播结果中的当前索引

`迭代器`

以`self`“组件”的迭代器元组。

`nd`

广播结果的维度数。

`维度`

广播结果的维度数。

`numiter`

广播结果拥有的迭代器数量。

`形状`

广播结果的形状。

`大小`

广播结果的总大小。

方法

| `重置`() | 重置广播结果的迭代器。 |
