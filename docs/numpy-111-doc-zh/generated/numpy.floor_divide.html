numpy.floor_divide(x1, x2[, out]) = <ufunc 'floor_divide'>

返回小于或等于输入的除法的最大整数。它等效于Python //运算符，并与Python %（remainder）函数配对，使b = a ％ b + t14> * （a // b）

参数：

x1：array_like

分子。

x2：array_like

分母。
返回：

y：ndarray

y = floor（x1 / x2）

也可以看看

remainder
剩余与floor_divide互补。
divide
标准部门。
floor
将数字向零舍入到最接近的整数。
ceil
将数字向最接近的整数向无穷大舍入。

例子

>>> np.floor_divide(7,3)
2
>>> np.floor_divide([1., 2., 3., 4.], 2.5)
array([ 0.,  0.,  1.,  1.])
