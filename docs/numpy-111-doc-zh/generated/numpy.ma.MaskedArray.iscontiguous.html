numpy.ma.MaskedArray.iscontiguous

MaskedArray.iscontiguous()[source]

返回一个布尔值，表示数据是否是连续的。

参数：无

例子

>>> x = np.ma.array([1, 2, 3])
>>> x.iscontiguous()
True

iscontiguous返回掩码数组的标志之一：

>>> x.flags
  C_CONTIGUOUS : True
  F_CONTIGUOUS : True
  OWNDATA : False
  WRITEABLE : True
  ALIGNED : True
  UPDATEIFCOPY : False
