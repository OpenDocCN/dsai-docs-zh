# pandas.Index.max

> 原文：[`pandas.pydata.org/docs/reference/api/pandas.Index.max.html`](https://pandas.pydata.org/docs/reference/api/pandas.Index.max.html)

```py
Index.max(axis=None, skipna=True, *args, **kwargs)
```

返回索引的最大值。

参数：

**axis**int, 可选

为了与 NumPy 兼容。仅允许 0 或 None。

**skipna**bool, 默认 True

在显示结果时排除 NA/null 值。

***args, **kwargs**

与 NumPy 兼容的附加参数和关键字。

返回：

标量

最大值。

另请参阅

`Index.min`

返回索引中的最小值。

`Series.max`

返回 Series 中的最大值。

`DataFrame.max`

返回 DataFrame 中的最大值。

示例

```py
>>> idx = pd.Index([3, 2, 1])
>>> idx.max()
3 
```

```py
>>> idx = pd.Index(['c', 'b', 'a'])
>>> idx.max()
'c' 
```

对于 MultiIndex，最大值是按词典顺序确定的。

```py
>>> idx = pd.MultiIndex.from_product([('a', 'b'), (2, 1)])
>>> idx.max()
('b', 2) 
```
