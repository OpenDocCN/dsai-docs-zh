numpy.core.defchararray.lower(a)[source]

返回一个数组，将元素转换为小写。

逐个调用str.lower。

对于8位字符串，此方法依赖于区域设置。

参数：

a：array_like，{str，unicode}

输入数组。
返回：

out：ndarray，{str，unicode}

输出str或unicode的数组，取决于输入类型

也可以看看

str.lower

例子

>>> c = np.array(['A1B C', '1BCA', 'BCA1']); c
array(['A1B C', '1BCA', 'BCA1'],
      dtype='|S5')
>>> np.char.lower(c)
array(['a1b c', '1bca', 'bca1'],
      dtype='|S5')
