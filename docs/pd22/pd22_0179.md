# pandas.Series.pop

> 原文：[`pandas.pydata.org/docs/reference/api/pandas.Series.pop.html`](https://pandas.pydata.org/docs/reference/api/pandas.Series.pop.html)

```py
Series.pop(item)
```

从系列中返回项目并删除。如果未找到，则引发 KeyError。

参数：

**item**标签

需要被移除的元素的索引。

返回：

从系列中弹出的值。

示例

```py
>>> ser = pd.Series([1, 2, 3]) 
```

```py
>>> ser.pop(0)
1 
```

```py
>>> ser
1    2
2    3
dtype: int64 
```
