numpy.typename

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.typename.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.typename(char)[source]

返回给定数据类型代码的描述。

参数：

char：str

数据类型代码。
返回：

out：str

输入数据类型代码的说明。

也可以看看

dtype，typecodes

例子

>>> typechars = ['S1', '?', 'B', 'D', 'G', 'F', 'I', 'H', 'L', 'O', 'Q',
...              'S', 'U', 'V', 'b', 'd', 'g', 'f', 'i', 'h', 'l', 'q']
>>> for typechar in typechars:
...     print(typechar, ' : ', np.typename(typechar))
...
S1  :  character
?  :  bool
B  :  unsigned char
D  :  complex double precision
G  :  complex long double precision
F  :  complex single precision
I  :  unsigned integer
H  :  unsigned short
L  :  unsigned long integer
O  :  object
Q  :  unsigned long long integer
S  :  string
U  :  unicode
V  :  void
b  :  signed char
d  :  double precision
g  :  long precision
f  :  single precision
i  :  integer
h  :  short
l  :  long integer
q  :  long long integer
