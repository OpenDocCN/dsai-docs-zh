MaskedArray.ptp(axis=None, out=None, fill_value=None)[source]

沿给定维度（即峰 - 峰值）的返回（最大 - 最小）。

参数：

axis：{None，int}，可选

用于查找峰的轴。如果为无（默认），则使用展平的数组。

out：{None，array_like}，可选

用于放置结果的替代输出数组。它必须具有与预期输出相同的形状和缓冲区长度，但如果需要，将转换类型。

fill_value：{var}，可选

用于填充屏蔽值的值。
返回：

ptp：ndarray。

除非指定out，否则保存结果的新数组将返回out的引用。
