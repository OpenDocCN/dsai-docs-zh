numpy.iscomplex(x)[source]

返回bool数组，其中如果输入元素很复杂，则返回True。

测试是输入是否具有非零虚部，而不是输入类型是否复杂。

参数：

x：array_like

输入数组。
返回：

out：ndarray的bools

输出数组。

也可以看看

isreal

iscomplexobj
如果x是复数类型或复数数组，则返回True。

例子

>>> np.iscomplex([1+1j, 1+0j, 4.5, 3, 2, 2j])
array([ True, False, False, False, False,  True], dtype=bool)
