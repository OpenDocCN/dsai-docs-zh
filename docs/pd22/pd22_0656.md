# pandas.Index.ndim

> 原文：[`pandas.pydata.org/docs/reference/api/pandas.Index.ndim.html`](https://pandas.pydata.org/docs/reference/api/pandas.Index.ndim.html)

```py
property Index.ndim
```

数据底层维度的数量，根据定义为 1。

例子

```py
>>> s = pd.Series(['Ant', 'Bear', 'Cow'])
>>> s
0     Ant
1    Bear
2     Cow
dtype: object
>>> s.ndim
1 
```

对于索引：

```py
>>> idx = pd.Index([1, 2, 3])
>>> idx
Index([1, 2, 3], dtype='int64')
>>> idx.ndim
1 
```
