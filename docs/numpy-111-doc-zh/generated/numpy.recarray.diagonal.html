校对：（虚位以待）

recarray.diagonal(offset=0, axis1=0, axis2=1)

返回指定的对角线。在NumPy 1.9中，返回的数组是一个只读视图，而不是像以前的NumPy版本一样的副本。在将来的版本中，只读限制将被删除。
