numpy.polynomial.laguerre.lagline(off, scl)[source]

Laguerre系列的图形是一条直线。

参数：

off，scl：标量

指定的行由关闭 + scl * x给出。
返回：

y：ndarray

此模块表示关 + scl * x的Laguerre系列。

也可以看看

polyline，chebline

例子

>>> from numpy.polynomial.laguerre import lagline, lagval
>>> lagval(0,lagline(3, 2))
3.0
>>> lagval(1,lagline(3, 2))
5.0
