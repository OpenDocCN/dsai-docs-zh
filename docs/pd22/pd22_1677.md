# `pandas.core.groupby.SeriesGroupBy.cumsum`

> 原文：[`pandas.pydata.org/docs/reference/api/pandas.core.groupby.SeriesGroupBy.cumsum.html`](https://pandas.pydata.org/docs/reference/api/pandas.core.groupby.SeriesGroupBy.cumsum.html)

```py
SeriesGroupBy.cumsum(axis=_NoDefault.no_default, *args, **kwargs)
```

每个组的累积和。

返回：

Series 或 DataFrame

另请参阅

`Series.groupby`

将函数应用于 Series 进行分组。

`DataFrame.groupby`

将函数应用于 DataFrame 的每一行或列进行分组。

示例

对于 SeriesGroupBy：

```py
>>> lst = ['a', 'a', 'b']
>>> ser = pd.Series([6, 2, 0], index=lst)
>>> ser
a    6
a    2
b    0
dtype: int64
>>> ser.groupby(level=0).cumsum()
a    6
a    8
b    0
dtype: int64 
```

对于 DataFrameGroupBy：

```py
>>> data = [[1, 8, 2], [1, 2, 5], [2, 6, 9]]
>>> df = pd.DataFrame(data, columns=["a", "b", "c"],
...                   index=["fox", "gorilla", "lion"])
>>> df
 a   b   c
fox       1   8   2
gorilla   1   2   5
lion      2   6   9
>>> df.groupby("a").groups
{1: ['fox', 'gorilla'], 2: ['lion']}
>>> df.groupby("a").cumsum()
 b   c
fox       8   2
gorilla  10   7
lion      6   9 
```
