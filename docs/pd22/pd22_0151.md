# pandas.Series.ndim

> [`pandas.pydata.org/docs/reference/api/pandas.Series.ndim.html`](https://pandas.pydata.org/docs/reference/api/pandas.Series.ndim.html)

```py
property Series.ndim
```

底层数据的维度数量，根据定义为 1。

示例

```py
>>> s = pd.Series(['Ant', 'Bear', 'Cow'])
>>> s
0     Ant
1    Bear
2     Cow
dtype: object
>>> s.ndim
1 
```

对于索引：

```py
>>> idx = pd.Index([1, 2, 3])
>>> idx
Index([1, 2, 3], dtype='int64')
>>> idx.ndim
1 
```
