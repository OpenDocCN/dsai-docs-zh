numpy.matrix.getT

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.matrix.getT.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

matrix.getT()[source]

返回矩阵的转置。

不共轭！对于复共轭转置，使用.H。

参数：

无
返回：

ret：matrix对象

矩阵的（非共轭）转置。

也可以看看

transpose，getH

例子

>>> m = np.matrix('[1, 2; 3, 4]')
>>> m
matrix([[1, 2],
        [3, 4]])
>>> m.getT()
matrix([[1, 3],
        [2, 4]])
