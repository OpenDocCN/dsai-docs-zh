HermiteE.mapparms()[source]

返回映射参数。

返回值定义应用于系列之前的输入参数的线性映射off + scl * x 。映射取决于domain和window；如果当前domain等于window，则生成的映射是身份。如果系列实例的系数要由该类外部使用，则线性函数必须替换基本多项式的标准表示中的x。

返回：

off，scl：float或complex

映射函数由off + scl * x定义。

笔记

如果当前域是间隔[l1， r1]并且窗口是[l2， r2]，则线性映射函数L由以下等式定义：

L(l1) = l2
L(r1) = r2
