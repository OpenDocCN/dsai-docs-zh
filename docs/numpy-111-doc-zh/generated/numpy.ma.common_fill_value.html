numpy.ma.common_fill_value(a, b)[source]

返回两个屏蔽数组的公共填充值（如果有）。

如果a.fill_value == b.fill_value，则返回填充值，否则返回None。

参数：

a，b：MaskedArray

用于比较填充值的蒙版数组。
返回：

fill_value：标量或无

公共填充值，或无。

例子

>>> x = np.ma.array([0, 1.], fill_value=3)
>>> y = np.ma.array([0, 1.], fill_value=3)
>>> np.ma.common_fill_value(x, y)
3.0
