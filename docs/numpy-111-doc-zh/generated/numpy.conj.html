numpy.conj

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.conj.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.conj(x[, out]) = <ufunc 'conjugate'>

按元素方式返回复共轭。

复数的复共轭通过改变其虚部的符号来获得。

参数：

x：array_like

输入值。
返回：

y：ndarray

x的复共轭，具有与y相同的dtype。

例子

>>> np.conjugate(1+2j)
(1-2j)

>>> x = np.eye(2) + 1j * np.eye(2)
>>> np.conjugate(x)
array([[ 1.-1.j,  0.-0.j],
       [ 0.-0.j,  1.-1.j]])
