dtype.names

字段名称的有序列表，如果没有字段，则为None。

这些名称根据增加的字节偏移量排序。这可以用于，例如，以偏移顺序遍历所有命名的字段。

例子

>>> dt = np.dtype([('name', np.str_, 16), ('grades', np.float64, (2,))])
>>> dt.names
('name', 'grades')
