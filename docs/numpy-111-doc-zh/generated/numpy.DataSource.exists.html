DataSource.exists(path)[source]

测试路径是否存在。

测试路径是否存在（按此顺序）：

  • 一个本地文件。
  • 已远程URL已下载并本地存储在DataSource目录中。
  • 未被下载但是有效且可访问的远程URL。
参数：

路径：str

可以是本地文件或远程URL。
返回：

out：bool

如果路径存在，则为真。

笔记

如果路径是网址，则exists将返回True，如果它存储在本地的DataSource目录中，或者是有效的远程URL。DataSource不区分两者，如果文件存在于任一位置，则可访问该文件。
