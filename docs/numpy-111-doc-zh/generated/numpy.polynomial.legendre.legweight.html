numpy.polynomial.legendre.legweight(x)[source]

勒让德多项式的权重函数。

权重函数为，积分间隔为。关于该权重函数，勒让德多项式是正交的，但不是归一化的。

参数：

x：array_like

将计算加权函数的值。
返回：

w：ndarray

在x处的权重函数。
