```py
scipy.special.nrdtrisd(p, x, mn, out=None) = <ufunc 'nrdtrisd'>
```

根据其他参数计算正态分布的标准差。

参数:

**p**array_like

CDF 值，范围在 (0, 1]。

**x**array_like

分位数，即积分的上限。

**mn**标量或ndarray

正态分布的平均值。

**out**ndarray，可选

可选输出数组以存储函数结果

返回:

**std**标量或ndarray

标准差。
