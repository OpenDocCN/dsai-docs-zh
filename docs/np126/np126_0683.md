column_stack

将 1-D 数组堆叠为 2-D 数组的列。

采用 1-D 数组的序列，并将它们堆叠为列，以形成单个 2-D 数组。2-D 数组按原样堆叠，就像使用`hstack`一样。首先将 1-D 数组转换为 2-D 列。

参数：

**tup**是 1-D 或 2-D 数组的序列。

数组堆叠。它们都必须具有相同的第一维。

返回：

**stacked**2-D 数组

由给定数组堆叠形成的数组。

另请参见

`stack`, `hstack`, `vstack`, `concatenate`

注释

如果有的话，该函数适用于 _data 和 _mask。

例子

```py
>>> a = np.array((1,2,3))
>>> b = np.array((2,3,4))
>>> np.column_stack((a,b))
array([[1, 2],
 [2, 3],
 [3, 4]]) 
```
