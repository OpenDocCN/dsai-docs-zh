```py
scipy.special.kelvin(x, out=None) = <ufunc 'kelvin'>
```

开尔文函数作为复数

参数：

**x**array_like

参数

**out** ndarray 元组，可选

函数值的可选输出数组

返回值：

**Be, Ke, Bep, Kep**4-元组，标量或ndarray

元组（Be, Ke, Bep, Kep）包含复数，表示在*x*处评估的实部和虚部开尔文函数及其导数。例如，kelvin(x)[0].real = ber x 和 kelvin(x)[0].imag = bei x，ker 和 kei 类似。
