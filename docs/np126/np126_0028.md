# `numpy.ndarray.ndim`

> [`numpy.org/doc/1.26/reference/generated/numpy.ndarray.ndim.html`](https://numpy.org/doc/1.26/reference/generated/numpy.ndarray.ndim.html)

属性

```py
ndarray.ndim
```

数组的维度数量。

示例

```py
>>> x = np.array([1, 2, 3])
>>> x.ndim
1
>>> y = np.zeros((2, 3, 4))
>>> y.ndim
3 
```
