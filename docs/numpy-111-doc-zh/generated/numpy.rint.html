校对：（虚位以待）

numpy.rint(x[, out]) = <ufunc 'rint'>

数组的圆形元素到最接近的整数。

参数：

x：array_like

输入数组。
返回：

out：ndarray或scalar

输出数组的形状和类型与x相同。

也可以看看

ceil，floor，trunc

例子

>>> a = np.array([-1.7, -1.5, -0.2, 0.2, 1.5, 1.7, 2.0])
>>> np.rint(a)
array([-2., -2., -0.,  0.,  2.,  2.,  2.])
