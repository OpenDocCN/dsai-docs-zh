numpy.multiply

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.multiply(x1, x2[, out]) = <ufunc 'multiply'>

逐元素乘法参数。

参数：

x1，x2：array_like

输入要乘的数组。
返回：

y：ndarray

x1和x2的乘积。如果x1和x2都是标量，则返回标量。

笔记

在数组广播方面等同于x1 * x2。

例子

>>> np.multiply(2.0, 4.0)
8.0

>>> x1 = np.arange(9.0).reshape((3, 3))
>>> x2 = np.arange(3.0)
>>> np.multiply(x1, x2)
array([[  0.,   1.,   4.],
       [  0.,   4.,  10.],
       [  0.,   7.,  16.]])
