numpy.core.defchararray.istitle(a)[source]

如果元素是一个已滴定的字符串并且至少有一个字符，则为每个元素返回true，否则返回false。

逐个调用str.istitle。

对于8位字符串，此方法依赖于区域设置。

参数：

a：array_like of str或unicode
返回：

out：ndarray

输出数组的bools
