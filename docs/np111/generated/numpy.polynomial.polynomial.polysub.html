numpy.polynomial.polynomial.polysub(c1, c2)[source]

从另一个中减去一个多项式。

返回两个多项式c1 - c2的差异。参数是从最低阶项到最高阶的系数序列，即[1,2,3]表示多项式1 + 2 * x + 3 * x ** 2。

参数：

c1，c2：array_like

1-D从低到高排序的多项式系数的数组。
返回：

out：ndarray

表示它们的差的系数。

也可以看看

polyadd，polymul，polydiv，polypow

例子

>>> from numpy.polynomial import polynomial as P
>>> c1 = (1,2,3)
>>> c2 = (3,2,1)
>>> P.polysub(c1,c2)
array([-2.,  0.,  2.])
>>> P.polysub(c2,c1) # -P.polysub(c1,c2)
array([ 2.,  0., -2.])
