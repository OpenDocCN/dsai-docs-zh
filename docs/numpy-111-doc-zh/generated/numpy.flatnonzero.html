numpy.flatnonzero(a)[source]

在a的扁平版本中返回非零的索引。

这相当于a.ravel()。nonzero()[0]。

参数：

a：ndarray

输入数组。
返回：

res：ndarray

输出数组，其包含非零的a.ravel()的元素的索引。

也可以看看

nonzero
返回输入数组的非零元素的索引。
ravel
返回包含输入数组元素的1-D数组。

例子

>>> x = np.arange(-2, 3)
>>> x
array([-2, -1,  0,  1,  2])
>>> np.flatnonzero(x)
array([0, 1, 3, 4])

使用非零元素的索引作为索引数组来提取这些元素：

>>> x.ravel()[np.flatnonzero(x)]
array([-2, -1,  1,  2])
