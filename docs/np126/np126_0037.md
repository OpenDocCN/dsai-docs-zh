# numpy.ndarray.imag

> 原文：[`numpy.org/doc/1.26/reference/generated/numpy.ndarray.imag.html`](https://numpy.org/doc/1.26/reference/generated/numpy.ndarray.imag.html)

属性

```py
ndarray.imag
```

数组的虚部。

示例

```py
>>> x = np.sqrt([1+0j, 0+1j])
>>> x.imag
array([ 0\.        ,  0.70710678])
>>> x.imag.dtype
dtype('float64') 
```
