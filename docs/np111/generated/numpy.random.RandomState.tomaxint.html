RandomState.tomaxint(size=None)

0和sys.maxint之间的随机整数，包括0和。

在区间[0，sys.maxint]中返回均匀分布的随机整数的样本。

参数：

size：int或tuple的整数，可选

输出形状。如果给定形状是例如（m， n， k），则 m * n * k默认值为None，在这种情况下返回单个值。
返回：

out：ndarray

绘制样本，形状大小。

也可以看看

randint
在整数的给定半开间隔上的均匀采样。
random_integers
在给定的整数闭合区间内进行均匀采样。

例子

>>> RS = np.random.mtrand.RandomState() # need a RandomState object
>>> RS.tomaxint((2,2,2))
array([[[1170048599, 1600360186],
        [ 739731006, 1947757578]],
       [[1871712945,  752307660],
        [1601631370, 1479324245]]])
>>> import sys
>>> sys.maxint
2147483647
>>> RS.tomaxint((2,2,2)) < sys.maxint
array([[[ True,  True],
        [ True,  True]],
       [[ True,  True],
        [ True,  True]]], dtype=bool)
