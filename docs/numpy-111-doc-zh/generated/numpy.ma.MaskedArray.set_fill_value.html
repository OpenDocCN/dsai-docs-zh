MaskedArray.set_fill_value(value=None)[source]

设置掩码数组的填充值。

参数：

value：标量，可选

新的填充值。默认值为None，在这种情况下，将使用基于数据类型的默认值。

也可以看看

ma.set_fill_value
等效功能。

例子

>>> x = np.ma.array([0, 1.], fill_value=-np.inf)
>>> x.fill_value
-inf
>>> x.set_fill_value(np.pi)
>>> x.fill_value
3.1415926535897931

重置为默认：

>>> x.set_fill_value()
>>> x.fill_value
1e+20
