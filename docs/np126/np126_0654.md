# numpy.ma.resize

> 原文：[`numpy.org/doc/1.26/reference/generated/numpy.ma.resize.html`](https://numpy.org/doc/1.26/reference/generated/numpy.ma.resize.html)

```py
ma.resize(x, new_shape)
```

返回一个具有指定大小和形状的新掩码数组。

这是 `numpy.resize` 函数的掩码版本。新数组将以 *x* 的重复副本填充（按照内存中的数据存储顺序）。如果 *x* 被掩码，则新数组将被掩码，并且新掩码将是旧掩码的重复。

另请参阅

`numpy.resize`

在顶级 NumPy 模块中的等效函数。

示例

```py
>>> import numpy.ma as ma
>>> a = ma.array([[1, 2] ,[3, 4]])
>>> a[0, 1] = ma.masked
>>> a
masked_array(
 data=[[1, --],
 [3, 4]],
 mask=[[False,  True],
 [False, False]],
 fill_value=999999)
>>> np.resize(a, (3, 3))
masked_array(
 data=[[1, 2, 3],
 [4, 1, 2],
 [3, 4, 1]],
 mask=False,
 fill_value=999999)
>>> ma.resize(a, (3, 3))
masked_array(
 data=[[1, --, 3],
 [4, 1, --],
 [3, 4, 1]],
 mask=[[False,  True, False],
 [False, False,  True],
 [False, False, False]],
 fill_value=999999) 
```

不管输入类型如何，都将返回一个 MaskedArray。

```py
>>> a = np.array([[1, 2] ,[3, 4]])
>>> ma.resize(a, (3, 3))
masked_array(
 data=[[1, 2, 3],
 [4, 1, 2],
 [3, 4, 1]],
 mask=False,
 fill_value=999999) 
```
