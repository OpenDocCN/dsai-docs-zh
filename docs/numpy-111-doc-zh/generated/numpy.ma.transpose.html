numpy.ma.transpose

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.ma.transpose.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.ma.transpose(a, axes=None)[source]

允许数组的尺寸。

此函数与numpy.transpose完全相同。

也可以看看

numpy.transpose
顶级NumPy模块中的等效函数。

例子

>>> import numpy.ma as ma
>>> x = ma.arange(4).reshape((2,2))
>>> x[1, 1] = ma.masked
>>>> x
masked_array(data =
 [[0 1]
 [2 --]],
             mask =
 [[False False]
 [False  True]],
       fill_value = 999999)
>>> ma.transpose(x)
masked_array(data =
 [[0 2]
 [1 --]],
             mask =
 [[False False]
 [False  True]],
       fill_value = 999999)
