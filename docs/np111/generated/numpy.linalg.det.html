numpy.linalg.det(a)[source]

计算数组的行列式。

参数：

a：（...，M，M）array_like

输入数组以计算行列式。
返回：

det：（...）array_like

a的决定因素。

也可以看看

slogdet
另一种表示行列式的方法，更适合于可能发生下溢/溢出的大矩阵。

笔记

版本1.8.0中的新功能。

广播规则适用，有关详细信息，请参阅numpy.linalg文档。

行列式通过使用LAPACK例程z / dgetrf的LU因式分解来计算。

例子

2-D数组[[a，b]，[c，d]]的行列式是ad-bc：

>>> a = np.array([[1, 2], [3, 4]])
>>> np.linalg.det(a)
-2.0

计算矩阵堆栈的决定因素：

>>> a = np.array([ [[1, 2], [3, 4]], [[1, 2], [2, 1]], [[1, 3], [3, 1]] ])
>>> a.shape
(3, 2, 2)
>>> np.linalg.det(a)
array([-2., -3., -8.])
