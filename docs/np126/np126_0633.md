# numpy.ma.getmaskarray

> 原文：[`numpy.org/doc/1.26/reference/generated/numpy.ma.getmaskarray.html`](https://numpy.org/doc/1.26/reference/generated/numpy.ma.getmaskarray.html)

```py
ma.getmaskarray(arr)
```

返回掩码数组的掩码，或全布尔数组。

如果 *arr* 是 `MaskedArray` 并且掩码不是 `nomask`，则返回掩码数组的掩码作为 ndarray，否则返回形状与 *arr* 相同的全布尔数组。

参数：

**arr**array_like

输入 `MaskedArray` 需要掩码的数组。

另请参阅

`getmask`

返回掩码数组的掩码，或者无掩码。

`getdata`

返回掩码数组的数据作为 ndarray。

示例

```py
>>> import numpy.ma as ma
>>> a = ma.masked_equal([[1,2],[3,4]], 2)
>>> a
masked_array(
 data=[[1, --],
 [3, 4]],
 mask=[[False,  True],
 [False, False]],
 fill_value=2)
>>> ma.getmaskarray(a)
array([[False,  True],
 [False, False]]) 
```

当掩码 == `nomask` 时的结果

```py
>>> b = ma.masked_array([[1,2],[3,4]])
>>> b
masked_array(
 data=[[1, 2],
 [3, 4]],
 mask=False,
 fill_value=999999)
>>> ma.getmaskarray(b)
array([[False, False],
 [False, False]]) 
```
