numpy.matlib.eye(n, M=None, k=0, dtype=<type 'float'>)[source]

返回一个矩阵，在对角线上的1和其他地方的零。

参数：

n：int

输出中的行数。

M：int，可选

输出中的列数，默认为n。

k：int，可选

对角线的索引：0表示主对角线，正值表示上对角线，负值表示下对角线。

dtype：dtype，可选

返回矩阵的数据类型。
返回：

I：matrix

A n x M矩阵，其中除了其值等于1的k对角线，所有元素都等于零。

例子

>>> import numpy.matlib
>>> np.matlib.eye(3, k=1, dtype=float)
matrix([[ 0.,  1.,  0.],
        [ 0.,  0.,  1.],
        [ 0.,  0.,  0.]])
