numpy.core.defchararray.startswith(a, prefix, start=0, end=None)[source]

返回True的布尔数组，其中a中的字符串元素以前缀开头，否则为False。

以元素方式调用str.startswith。

参数：

a：array_like of str或unicode

前缀：str

start，end：int，可选

使用可选的开始，从该位置开始测试。使用可选的结束，停止在该位置的比较。
返回：

out：ndarray
