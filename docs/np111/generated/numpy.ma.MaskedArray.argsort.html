MaskedArray.argsort(axis=None, kind='quicksort', order=None, fill_value=None)[source]

返回沿指定轴对数组进行排序的索引数组。屏蔽值预先填充到fill_value。

参数：

axis：int，可选

要排序的轴。默认值为-1（最后一个轴）。如果为“无”，则使用展平的数组。

fill_value：var，可选

用于在排序之前填充数组的值。默认值为输入数组的fill_value属性。

kind：{'quicksort'，'mergesort'，'heapsort'}，可选

排序算法。

订单：list，可选

当a是定义了字段的数组时，此参数指定首先，第二等比较哪些字段。不需要指定所有字段。
返回：

index_array：ndarray，int

沿指定轴排序a的指数数组。换句话说，a[index_array]产生一个排序的a。

例子

>>> a = np.ma.array([3,2,1], mask=[False, False, True])
>>> a
masked_array(data = [3 2 --],
             mask = [False False  True],
       fill_value = 999999)
>>> a.argsort()
array([1, 0, 2])
