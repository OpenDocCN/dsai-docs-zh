numpy.issubdtype(arg1, arg2)[source]

返回True如果第一个参数是类型层次结构中的类型代码较低/相等。

参数：

arg1，arg2：dtype_like

dtype或表示类型代码的字符串。
返回：

out：bool

也可以看看

issubsctype，issubclass_

numpy.core.numerictypes
numpy类型层次结构概述。

例子

>>> np.issubdtype('S1', str)
True
>>> np.issubdtype(np.float64, np.float32)
False
