numpy.rot90(m, k=1)[source]

将数组沿逆时针方向旋转90度。

前两个维度旋转；因此，数组必须至少是2-D。

参数：

m：array_like

数组的两个或多个维度。

k：integer

数组旋转90度的次数。
返回：

y：ndarray

旋转数组。

也可以看看

fliplr
水平翻转数组。
flipud
垂直翻转数组。

例子

>>> m = np.array([[1,2],[3,4]], int)
>>> m
array([[1, 2],
       [3, 4]])
>>> np.rot90(m)
array([[2, 4],
       [1, 3]])
>>> np.rot90(m, 2)
array([[4, 3],
       [2, 1]])
