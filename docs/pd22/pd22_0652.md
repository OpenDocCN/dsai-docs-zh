# pandas.Index.shape

> [`pandas.pydata.org/docs/reference/api/pandas.Index.shape.html`](https://pandas.pydata.org/docs/reference/api/pandas.Index.shape.html)

```py
property Index.shape
```

返回基础数据的形状的元组。

示例

```py
>>> idx = pd.Index([1, 2, 3])
>>> idx
Index([1, 2, 3], dtype='int64')
>>> idx.shape
(3,) 
```
