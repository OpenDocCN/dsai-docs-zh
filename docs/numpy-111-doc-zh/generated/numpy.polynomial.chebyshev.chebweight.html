切比雪夫多项式的加权函数。

权重函数为，积分间隔为。相对于该权重函数，切比雪夫多项式是正交的，但不是正规化的。

参数：

x：array_like

将计算加权函数的值。
返回：

w：ndarray

在x处的权重函数。
