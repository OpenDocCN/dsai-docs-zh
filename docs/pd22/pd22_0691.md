# pandas.Index.nunique

> 原文：[`pandas.pydata.org/docs/reference/api/pandas.Index.nunique.html`](https://pandas.pydata.org/docs/reference/api/pandas.Index.nunique.html)

```py
Index.nunique(dropna=True)
```

返回对象中唯一元素的数量。

默认情况下排除 NA 值。

参数：

**dropna** 布尔值，默认为 True

不包括 NaN 在内的计数。

返回：

整数
