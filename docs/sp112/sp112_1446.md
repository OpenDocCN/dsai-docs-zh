SciPy 1.2.1 是一个纯粹的 bug 修复版本，与 1.2.0 相比没有新功能。最重要的是，它解决了因非 ASCII 字符问题导致无法在 Python 2.7 上从源代码安装 1.2.0 的问题。

还值得注意的是 SciPy 1.2.1 wheels 使用了 OpenBLAS 0.3.5.dev 构建，这可能缓解了在 SciPy 1.2.0 中观察到的一些线性代数问题。
