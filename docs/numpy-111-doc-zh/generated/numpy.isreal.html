numpy.isreal

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.isreal(x)[source]

返回bool数组，其中如果输入元素是实数，则返回True。

如果元素具有带零复杂部分的复杂类型，则该元素的返回值为True。

参数：

x：array_like

输入数组。
返回：

out：ndarray，bool

与x形状相同的布尔数组。

也可以看看

iscomplex

isrealobj
如果x不是复杂类型，则返回True。

例子

>>> np.isreal([1+1j, 1+0j, 4.5, 3, 2, 2j])
array([False,  True,  True,  True,  True, False], dtype=bool)
