numpy.broadcast_to(array, shape, subok=False)[source]

将数组广播到新形状。

参数：

数组：array_like

数组广播。

shape：tuple

所需数组的形状。

subok：bool，可选

如果为True，则子类将被传递，否则返回的数组将被强制为基类数组（默认）。
返回：

broadcast：数组

具有给定形状的原始数组的只读视图。它通常不连续。此外，广播数组的多于一个元件可以指单个存储器位置。
上升：

ValueError

如果数组根据NumPy的广播规则与新形状不兼容。

笔记

版本1.10.0中的新功能。

例子

>>> x = np.array([1, 2, 3])
>>> np.broadcast_to(x, (3, 3))
array([[1, 2, 3],
       [1, 2, 3],
       [1, 2, 3]])
