numpy.left_shift(x1, x2[, out]) = <ufunc 'left_shift'>

将整数的位向左移位。

通过在x1右侧添加x2 0，位向左移动。由于数字的内部表示是二进制格式，因此该操作等效于x1乘以2**x2。

参数：

x1：array_like为整数类型

输入值。

x2：array_like为整数类型

附加到x1的零数。必须是非负数。
返回：

out：整数类型的数组

返回x1，位向左移动x2次。

也可以看看

right_shift
将整数的位向右移位。
binary_repr
返回输入号码的二进制表示为字符串。

例子

>>> np.binary_repr(5)
'101'
>>> np.left_shift(5, 2)
20
>>> np.binary_repr(20)
'10100'

>>> np.left_shift(5, [1,2,3])
array([10, 20, 40])
