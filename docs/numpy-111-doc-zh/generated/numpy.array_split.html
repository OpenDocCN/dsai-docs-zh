numpy.array_split(ary, indices_or_sections, axis=0)[source]

将数组拆分为多个子数组。

请参阅split文档。这些函数之间的唯一区别是array_split允许indices_or_sections是不等分轴的整数。

也可以看看

split
将数组拆分成等大小的多个子数组。

例子

>>> x = np.arange(8.0)
>>> np.array_split(x, 3)
    [array([ 0.,  1.,  2.]), array([ 3.,  4.,  5.]), array([ 6.,  7.])]
