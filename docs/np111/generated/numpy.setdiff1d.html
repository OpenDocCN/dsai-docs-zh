numpy.setdiff1d(ar1, ar2, assume_unique=False)[source]

找到两个数组的设置差。

返回ar1中不在ar2中的排序，唯一值。

参数：

ar1：array_like

输入数组。

ar2：array_like

输入比较数组。

assume_unique：bool

如果为真，则输入数组都被假定为唯一的，这可以加速计算。默认值为False。
返回：

setdiff1d：ndarray

对ar1中不在ar2中的1D数值组进行排序。

也可以看看

numpy.lib.arraysetops
具有用于对数组执行设置操作的多个其他功能的模块。

例子

>>> a = np.array([1, 2, 3, 2, 4, 1])
>>> b = np.array([3, 4, 5, 6])
>>> np.setdiff1d(a, b)
array([1, 2])
