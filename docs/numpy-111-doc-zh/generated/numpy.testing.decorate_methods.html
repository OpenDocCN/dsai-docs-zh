numpy.testing.decorate_methods(cls, decorator, testmatch=None)[source]

对与正则表达式匹配的类中的所有方法应用装饰器。

给定的装饰器应用于由正则表达式testmatch（testmatch.search(methodname)）匹配的cls的所有公共方法。私有方法，即以下划线开头的方法将被忽略。

参数：

cls：class

装饰的方法的类。

装饰器：function

应用于方法的装饰器

testmatch：编译的regexp或str，可选

正则表达式。Default value is None, in which case the nose default (re.compile(r'(?:^|[\b_\.%s-])[Tt]est' % os.sep)) is used. 如果testmatch是一个字符串，它将首先编译为正则表达式。
