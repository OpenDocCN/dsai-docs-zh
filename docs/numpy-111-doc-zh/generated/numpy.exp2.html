numpy.exp2(x[, out]) = <ufunc 'exp2'>

对于输入数组中的所有p，计算2 ** p。

参数：

x：array_like

输入值。

out：ndarray，可选

数组将结果插入。
返回：

out：ndarray

元素方式2到幂x。
