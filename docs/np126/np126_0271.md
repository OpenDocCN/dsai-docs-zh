# `numpy.ufunc.nin`

> 原文：[`numpy.org/doc/1.26/reference/generated/numpy.ufunc.nin.html`](https://numpy.org/doc/1.26/reference/generated/numpy.ufunc.nin.html)

属性

```py
ufunc.nin
```

输入的数量。

数据属性，包含 ufunc 将作为输入处理的参数数量。

示例

```py
>>> np.add.nin
2
>>> np.multiply.nin
2
>>> np.power.nin
2
>>> np.exp.nin
1 
```
