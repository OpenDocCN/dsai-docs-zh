Making proper sets

unique（ar [，return_index，return_inverse，...]） 查找数组的唯一元素。

Boolean operations

in1d（ar1，ar2 [，assume_unique，invert]） 测试1-D数组的每个元素是否也存在于第二个数组中。
intersect1d(ar1, ar2[, assume_unique]) 找到两个数组。
setdiff1d（ar1，ar2 [，assume_unique]） 找到两个数组的设置差。
setxor1d（ar1，ar2 [，assume_unique]） 查找集的异或两个数组。
union1d（ar1，ar2） 查找两个数组的并集。
