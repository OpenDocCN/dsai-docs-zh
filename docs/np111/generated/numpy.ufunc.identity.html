ufunc.identity

身份值。

数据属性包含ufunc的标识元素，如果它有一个。如果没有，属性值为None。

例子

>>> np.add.identity
0
>>> np.multiply.identity
1
>>> np.power.identity
1
>>> print(np.exp.identity)
None
