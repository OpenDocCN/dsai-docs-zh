numpy.isscalar

numpy.isscalar(num)[source]

如果num的类型是标量类型，则返回True。

参数：

num：any

输入参数，可以是任何类型和形状。
返回：

val：bool

如果num是标量类型，则为True，否则为False。

例子

>>> np.isscalar(3.1)
True
>>> np.isscalar([3.1])
False
>>> np.isscalar(False)
True
