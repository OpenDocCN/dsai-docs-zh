Hermite.truncate(size)[source]

将序列截断到长度大小。

通过丢弃高度项将系列减少到长度size。size的值必须为正整数。这在最小二乘中是有用的，其中高次项的系数可以非常小。

参数：

size：positive int

通过舍弃高度项将该系列减小到长度大小。size的值必须为正整数。
返回：

new_series：series

截断系数的新系列实例。
