预计算特征值的椭球体径向函数 `obl_rad2`。

计算椭球体径向函数的第二类及其导数（相对于 *x*），适用于模式参数 m>=0 和 n>=m，椭球体参数 *c* 和 `|x| < 1.0`。需要预先计算好的特征值。

参数：

**m** array_like

模式参数 m (非负)

**n** array_like

模式参数 n (>= m)

**c** array_like

椭球体参数

**cv** array_like

特征值

**x** array_like

参数 x (`|x| < 1.0`)

**out** ndarray, 可选

可选的输出数组用于函数结果。

返回值：

**s** 标量或者 ndarray

函数值

**sp** 标量或者 ndarray

导数值 vs x
