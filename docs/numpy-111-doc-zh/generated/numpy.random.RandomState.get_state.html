RandomState.get_state()

返回一个元组，表示生成器的内部状态。

有关详细信息，请参见set_state。

返回：

out : tuple(str, ndarray of 624 uints, int, int, float)

返回的元组有以下项：

  1. 字符串'MT19937'。
  2. 624个无符号整数键的1-D数组。
  3. 整数pos。
  4. 整数has_gauss。
  5. 浮点数cached_gaussian。

另见

set_state

笔记

不需要set_state和get_state来处理NumPy中的任何随机分布。如果内部状态被手动改变，则用户应该准确地知道他/她正在做什么。
