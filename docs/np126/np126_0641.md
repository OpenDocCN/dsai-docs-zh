# numpy.ma.isMA

> 原文：[`numpy.org/doc/1.26/reference/generated/numpy.ma.isMA.html`](https://numpy.org/doc/1.26/reference/generated/numpy.ma.isMA.html)

```py
ma.isMA(x)
```

测试输入是否为 MaskedArray 的实例。

如果 *x* 是 MaskedArray 的实例，则返回 True，否则返回 False。任何对象都可接受为输入。

参数：

**x**object

要测试的对象。

返回：

**result**bool

如果 *x* 是 MaskedArray，则返回 True。

另请参阅

`isMA`

别名为 isMaskedArray。

`isarray`

别名为 isMaskedArray。

示例

```py
>>> import numpy.ma as ma
>>> a = np.eye(3, 3)
>>> a
array([[ 1.,  0.,  0.],
 [ 0.,  1.,  0.],
 [ 0.,  0.,  1.]])
>>> m = ma.masked_values(a, 0)
>>> m
masked_array(
 data=[[1.0, --, --],
 [--, 1.0, --],
 [--, --, 1.0]],
 mask=[[False,  True,  True],
 [ True, False,  True],
 [ True,  True, False]],
 fill_value=0.0)
>>> ma.isMaskedArray(a)
False
>>> ma.isMaskedArray(m)
True
>>> ma.isMaskedArray([0, 1, 2])
False 
```
