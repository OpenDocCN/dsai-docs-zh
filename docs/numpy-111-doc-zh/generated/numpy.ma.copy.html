numpy.ma.copy

原文：https://docs.scipy.org/doc/numpy/reference/generated/numpy.ma.copy.html

译者：飞龙 UsyiyiCN

校对：（虚位以待）

numpy.ma.copy(self, *args, **params) a.copy(order='C') = <numpy.ma.core._frommethod instance>
返回数组的副本。
参数：

订单：{'C'，'F'，'A'，'K'}，可选

控制副本的内存布局。'C'表示C阶，'F'表示F阶，'A'表示如果a是Fortran连续的'F'，否则为'C'。'K'表示尽可能接近a的布局。（请注意，此函数和：func：numpy.copy非常相似，但它们的order = arguments具有不同的默认值。）

例子

>>> x = np.array([[1,2,3],[4,5,6]], order='F')

>>> y = x.copy()

>>> x.fill(0)

>>> x
array([[0, 0, 0],
       [0, 0, 0]])

>>> y
array([[1, 2, 3],
       [4, 5, 6]])

>>> y.flags['C_CONTIGUOUS']
True
