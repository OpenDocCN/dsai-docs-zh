ndarray.flat

数组上的1-D迭代器。

这是一个numpy.flatiter实例，它类似于Python的内置迭代器对象，但不是其子类。

也可以看看

flatten
将折叠的数组的副本返回到一个维度。

flatiter

例子

>>> x = np.arange(1, 7).reshape(2, 3)
>>> x
array([[1, 2, 3],
       [4, 5, 6]])
>>> x.flat[3]
4
>>> x.T
array([[1, 4],
       [2, 5],
       [3, 6]])
>>> x.T.flat[3]
5
>>> type(x.flat)
<type 'numpy.flatiter'>

分配示例：

>>> x.flat = 3; x
array([[3, 3, 3],
       [3, 3, 3]])
>>> x.flat[[1,4]] = 1; x
array([[3, 1, 3],
       [3, 1, 3]])
